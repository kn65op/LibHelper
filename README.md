Project use cmake.
To build create directory and go to it:
- cmake ../

Build parameters:
- CMAKE_BUILD_TYPE - [required] can be DEBUG or RELEASE, specyfies build type
- BUILDS_TESTS - [optional] to build UT set to Yes
